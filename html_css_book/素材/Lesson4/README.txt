Lesson4のサンプルデータについて
--------------------------------------------------------------------------------

【ご注意】

（1）XDのバージョンについて
お使いのXDバージョンが古い場合、デザインカンプファイルが開けない可能性があります。
バージョンが古くてファイルが開けない場合は、指示にしたがって最新版にアップデートをしてください。

（2）使用フォントについて
このデザインカンプファイルには、Google FontsのWebフォントが使用されています。
フォントをインストールせずに表示した場合、レイアウトが崩れる場合がありますのでご注意ください。
正しく表示するため、各自でGoogle Fontsから以下のフォントをダウンロードして、ご自分のマシンにインストールしてから表示するようにしてください。

▼Google Fonts
https://fonts.google.com/?subset=japanese
・Noto Sans JP Regular
・Noto Sans JP Bold

（3）写真素材について
このサンプルに使用されている写真は、以下のコワーキングスペースの協力を得て撮影させていただいたものです。サンプル内で使用されている写真素材の転載・再配布・二次利用は固くお断りさせていただきます。

撮影協力：コワーキングスペース『base Co+』（https://coplus.space/）
カメラマン：木澤 尚之

（4）完成サンプルデータについて
書籍ではデザインカンプからのコーディング設計部分のみが対象となっているため、
それに基づいた実際のコード解説は行っておりません。
コード内容に対する質問は原則としてお受けしておりませんのでご了承ください。
